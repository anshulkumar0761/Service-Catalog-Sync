# Service-Catalog-Sync
## 📌 Project Overview
The **Software Request Workflow** is a streamlined system that demonstrates how employees in an organization can request the installation of licensed software through a standard service catalog. The system maps the end-to-end workflow between employees, managers, and the IT department to ensure compliance and control.

## 🎯 Purpose of the Project
The primary goal of this project is to model and understand how organizations manage software installation requests. It provides a structured process to ensure that only authorized and correctly licensed software is deployed on company assets.

## 🚀 Key Features
* **Request Submission:** Simple form for employees to submit a request.
* **Approval Workflow:** Request automatically routes to the appropriate manager for approval.
* **IT Review:** Final review by the IT team for security and compatibility.
* **Installation:** Guided installation process for the employee upon approval.
